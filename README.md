# AI Bootcamp Divar Project

##  Overview
This project analyzes the real estate market in iran using a dataset of **1,000,000+ ads**.
Our goal is to identify price trends, locate luxury hotspots, and build a Machine Learning model to predict housing prices.

## Key Features 
- **Data Cleaning**: Handling missing values and removing outliers from a messy dataset.
- **Visualization**:
  - Interactive Heatmaps of Price & Density.
  - Price Trends over 2024-2025.
  - Correlation Matrix of features.
- **Statistical Analysis**: T-Test validation for Metropolis prices.
